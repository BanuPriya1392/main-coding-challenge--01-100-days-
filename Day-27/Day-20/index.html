Welcome to the navigation drawer! Click the menu button in the header to open the navigation drawer. You can also click outside the drawer or press the ESC key to close it.

This drawer is built using pure JavaScript, HTML, and CSS. It features smooth animations, responsive design, and a clean modern interface with an olive green color theme.

Features:

  • Smooth slide-in animation
  • Click outside to close
  • ESC key support
  • Responsive design
  • Active page highlighting
  • Accessible navigation
  • Olive green color theme
